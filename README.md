# Mini Katalog

Flutter ile yazılmış ufak bir ürün katalog ve sepet uygulaması.

Ürünleri listeleyip arayabilir, detayını görüp sepete ekleyebilirsin.

## Çalıştırmak için

```bash
flutter pub get
flutter run
```

Dart 3.0 ve üzeri gerekiyor.

## Klasörler

- `lib/main.dart` — uygulama girişi ve rotalar
- `lib/models/` — Product modeli
- `lib/pages/` — home, ürün detay ve sepet ekranları
- `lib/services/` — sepet ve örnek ürün verisi

Ekran görüntüleri `screenshots/` klasöründe.
